## 核心功能

### 1. 智能搜索

**适用场景：** 快速获取某个主题的综合信息

### 2. 市场尽调

**适用场景：** 为企业出海决策提供专业的市场尽职调查报告

#### 如何使用

1. 点击输入框上方的 **"尽调"（Market Due Diligence）** 按钮
2. 系统会自动弹出信息收集表单，填写以下信息：
   - **目标国家/区域**：例如：美国、东南亚、欧洲
   - **公司名称**：您的公司名称
   - **产品或服务**：核心产品或服务描述
   - **归属行业**：例如：科技、制造业、消费品
   - **其他具体需求**（可选）：特殊的分析需求
3. 点击提交，系统将自动生成专业的市场尽调报告

#### 报告结构

生成的报告包含以下五大部分：

##### 📊 第一部分：国别研究（PEST分析）
- **Political（政治）**：政治稳定性、双边关系、FDI政策
- **Economic（经济）**：经济指标、商业基础设施
- **Social（社会）**：人口统计、消费者行为、品牌认知
- **Technological（技术）**：技术采用度、数字生态成熟度

##### 🏭 第二部分：行业研究（SMART模型）
- **Scale（规模）**：市场规模、增长趋势、预测
- **Structure（结构）**：市场集中度、价值链分析
- **Model（商业模式）**：主流商业模式分析
- **Assets（资产）**：成功关键因素
- **Rule（潜规则）**：商业习俗、关系网络
- **Regulation（明规则）**：行业法规、准入标准
- **Technology（技术）**：核心技术、标准、变革速度

##### 🎯 第三部分：出海标杆研究
分析2-3家标杆企业（中国出海企业、全球竞品、本地龙头），每家包含：
1. **发展历史**：关键里程碑、战略转型
2. **主营产品**：产品组合、差异化定位
3. **商业模式**：收入模式、成本结构
4. **财务数据**：营收、增长率、盈利能力
5. **股东结构**：主要股东、控制权结构
6. **管理团队**：高管背景、团队优势
7. **资本运作**：融资历史、IPO、并购活动

##### 🚀 第四部分：出海路径规划（七步法）
1. **Entry Strategy（进入策略）**：推荐的市场进入模式
2. **Product Adaptation（产品适配）**：本地化要求、MVP方法
3. **Go-to-Market（市场营销）**：目标客群、定位、渠道
4. **Human Resources（人力资源）**：本地团队建设
5. **Capital Planning（资本规划）**：投资预算、融资里程碑
6. **Compliance & Risk（合规风险）**：前5大风险及应对
7. **Deep Localization（深度本地化）**：长期本地化战略

##### 📚 第五部分：参考资料
- 完整的数据来源列表
- 所有引用的URL链接
- 按类别分组（市场数据、竞争分析、法规政策等）

#### 自动模型优化

⚡ **智能功能**：当您选择"市场尽调"模式时，系统会自动将AI模型切换到 **Claude Sonnet 4.5**，这是目前最适合生成长篇专业报告的模型，确保报告质量和深度。

#### 导出和分享

生成的报告支持：
- 📄 导出为PDF格式
- 📝 导出为Word文档（.docx）
- 🔗 生成分享链接
- 💾 保存到报告库

### 3. 深度研究

**适用场景：** 对某个主题进行更深入、更全面的研究分析

#### 如何使用

1. 点击输入框上方的 **"深研"（Deep Research）** 按钮
2. 输入您想要深入研究的主题
3. 系统会进行多轮搜索、交叉验证，生成更详细的研究报告

#### 与智能搜索的区别

| 功能 | 智能搜索 | 深度研究 |
|------|----------|----------|
| 响应速度 | 快速（10-30秒） | 较慢（1-5分钟） |
| 搜索深度 | 基础检索 | 多轮深度检索 |
| 数据来源数量 | 5-15个 | 30-50+个 |
| 交叉验证 | 基础验证 | 多源交叉验证 |
| 报告长度 | 简要概述 | 详细分析 |
| 适用场景 | 快速了解 | 深入研究 |

## 常见问题

### Q1: 是否需要注册账号才能使用？

**A:** 不需要。NaviX支持访客模式，所有核心功能（搜索、市场尽调、深度研究）都可以在未登录状态下使用。

注册账号的好处：
- ✅ 保存聊天历史
- ✅ 收藏和管理报告
- ✅ 分享研究成果
- ✅ 跨设备同步

### Q2: 市场尽调报告生成需要多长时间？

**A:** 通常需要 **1-3分钟**，具体取决于：
- 目标市场的数据丰富程度
- 行业复杂度
- 当前系统负载

报告生成过程中，您可以看到实时进度指示。

### Q3: 生成的报告可以导出吗？

**A:** 可以。报告支持以下导出格式：
- **PDF** - 保留完整格式和图表
- **Word (.docx)** - 方便进一步编辑
- **分享链接** - 生成在线阅读链接

### Q4: 搜索结果的数据来源是什么？

**A:** NaviX使用多个高质量数据源：
- **Tavily AI** - 综合搜索引擎（默认）
- **Exa** - 神经搜索引擎
- **SearXNG** - 自托管搜索引擎
- 以及其他专业数据库

所有引用都会在回答中标注来源链接。

### Q5: 为什么有时候看到"自动选择Claude Sonnet 4.5"的提示？

**A:** 当您启用"市场尽调"模式时，系统会自动将模型切换到Claude Sonnet 4.5。这是因为：
- Claude Sonnet 4.5在生成长篇专业报告方面表现最佳
- 支持更长的输出（最多16,384 tokens）
- 推理能力和分析深度更强

这个自动切换确保了报告质量，即使您之前选择了其他模型。

### Q6: 如何提高搜索结果的准确性？

**提示技巧：**

❌ 不好的提问：
```
"美国市场"
"电动汽车"
```

✅ 好的提问：
```
"分析2024年美国电动汽车市场的规模、增长率和主要竞争格局"
"总结特斯拉在美国市场的销售策略和市场份额变化"
```

**最佳实践：**
1. 明确指定时间范围（如"2024年"、"最近3年"）
2. 明确指定地理区域
3. 使用具体的行业术语
4. 说明您想要的输出格式（分析、总结、对比等）

### Q7: 报告内容不够详细怎么办？

**解决方案：**

1. **使用追问功能** - 针对报告某个部分要求更详细的分析
   ```
   "请详细展开第三部分中关于竞争对手A的商业模式分析"
   ```

2. **切换到深度研究模式** - 对单个主题进行更深入的研究

3. **提供更多背景信息** - 在填写尽调表单时，在"其他具体需求"中说明您需要的详细程度

### Q8: 可以用中文提问吗？

**A:** 完全支持！NaviX支持：
- ✅ 中文提问
- ✅ 英文提问
- ✅ 中英文混合提问

市场尽调报告默认以简体中文输出，但可以在"其他需求"中要求英文报告。

### Q9: 数据的时效性如何？

**A:**
- **搜索功能**：实时抓取最新网络数据
- **市场尽调**：包含最近12个月内的新闻和数据
- **AI模型知识截止日期**：
  - Claude Sonnet 4.5: 2025年1月
  - GPT-4.1: 2024年12月
  - Gemini 2.5 Pro: 2025年3月

对于需要最新数据的问题，建议启用"搜索"模式。

### Q10: 如何分享我的研究报告？

**分享步骤：**

1. 在报告页面点击 **"分享"** 按钮
2. 系统生成唯一的分享链接
3. 复制链接发送给同事或合作伙伴
4. 接收者无需登录即可查看报告

**隐私控制：**
- 分享链接可以随时撤销
- 只有持有链接的人才能访问
- 不支持编辑，仅供阅读

## 高级功能

### 1. 自定义搜索深度

在搜索模式下，您可以通过提示词控制搜索深度：

```
"请进行基础搜索..." → 快速检索（5-10个来源）
"请进行深度搜索..." → 详细检索（15-30个来源）
```

### 2. 多语言支持

市场尽调功能会自动使用多语言搜索：
- **英文** - 国际市场资料
- **中文** - 中国企业背景资料
- **目标市场语言** - 本地化信息（如日语、德语、韩语等）

这确保了信息的全面性和准确性。

### 3. 连续对话

NaviX支持上下文记忆，您可以：

```
用户: "分析美国SaaS市场"
AI: [提供分析]

用户: "那欧洲市场呢？"  ← 系统理解"欧洲SaaS市场"
AI: [提供欧洲分析]

用户: "对比这两个市场的差异"  ← 系统理解是美国vs欧洲
AI: [提供对比分析]
```

### 4. 批量分析

如果您需要分析多个市场或多家公司，可以：

1. 在同一对话中连续提问
2. 要求系统生成对比表格
3. 导出完整的对话记录

示例：
```
"请依次分析以下三家公司的商业模式：公司A、公司B、公司C"
```

### 5. 设置浏览器搜索引擎

您可以将NaviX设置为浏览器的默认搜索引擎：

**步骤（以Chrome为例）：**
1. 打开浏览器设置
2. 搜索"搜索引擎"设置
3. 点击"管理搜索引擎和网站搜索"
4. 在"网站搜索"下点击"添加"
5. 填写：
   - 搜索引擎：NaviX
   - 快捷词：navix
   - 网址：`https://navix.moments.top/search?q=%s`
6. 保存后可以设置为默认搜索引擎

- 🚫 **不共享**：您的数据不会被分享给第三方
- ⚙️ **数据控制**：您可以随时删除自己的聊天历史和报告

### 使用AI模型的注意事项

- AI生成的报告是基于公开数据的分析，建议结合实地调研
- 财务数据和统计数字应当交叉验证
- 涉及法律法规的内容，建议咨询专业律师
- 商业决策应综合考虑多方面因素

## 最佳实践总结

### ✅ 推荐做法

1. **明确您的目标** - 清楚您想要回答的核心问题
2. **选择合适的模式** - 快速了解用搜索，深度分析用尽调
3. **提供详细背景** - 特别是在市场尽调模式下
4. **迭代式提问** - 先获取概览，再针对性深入
5. **验证关键数据** - 交叉检查重要的数字和结论
6. **保存重要结果** - 及时导出和保存有价值的报告
7. **善用追问** - 连续对话更高效

### ❌ 避免的做法

1. ❌ 问题过于宽泛（"分析美国市场"）
2. ❌ 缺少时间范围（"市场增长率"而不是"2024年市场增长率"）
3. ❌ 期望AI替代实地调研
4. ❌ 不验证关键数据就直接使用
5. ❌ 一次提问包含太多子问题
6. ❌ 频繁切换模式导致上下文丢失
